# Estilos CSS do projeto

Confira alguns detalhes quanto ao uso de cada um dos arquivos de estilo CSS no projeto


## Funcionalidade de cada arquivo
| nome do arquivo | local de uso | descrição |
|-----------------|--------------|-----------|
|main.css | pagina_principal.php |	Arquivo utilizado somente na página principal, utilizando Flexbox para melhor organização visual|
|cadastro.css | pagina_cadastro.php, pagina_editar_perfil.php, pagina_encerrar_conta.php, pagina_esqueci_senha | Arquivo usado em todas as páginas que contém formulários, com o objetivo de manter um estilo padronizado
|cor_mensagens.css|	todas as páginas|	Arquivo utilizado para padronizar as cores das mensagens de aviso ou erro no site
